This folder, 'exercises-X', it's a template for creating more exercises.
To create a new exercise follow this steps:

  • 1. Duplicate this folder.
  • 2. Add link into the main index file.
  • 3. Change the 'X' to the number of the exercise.
    1. In the folder name.
    2. In the paths to this folder name as the main index.
    3. In the 'title' of this html file.
    4. In the 'h1' of this html file.
    5. In the 'console.log' of ts file.
  • 4. Remove this instructions.
